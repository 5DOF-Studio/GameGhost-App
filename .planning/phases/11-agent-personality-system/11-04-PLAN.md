# Plan 11-04: Compose Leroy Personality — From Builder Questionnaire to Code

Using the completed Leroy personality builder questionnaire answers, compose rich personality content into the structured const strings in Agent.cs. Replace the placeholder sections from Plan 11-01 with fully authored personality that persists across voice, brain, and chat outputs.

## Prerequisites
- Plan 11-01 complete (personality architecture in place)
- **Owner has completed the Leroy section of the builder questionnaire** (`Agentic Office/RemoteAgents/Gaimer-Desktop-Brain/agent-personality-builder-questionnaire.md`)

## Tasks

### Task 1: Compose LeroySoul const string
From the builder answers, write the SOUL section (~200 tokens):
- Core identity (who Leroy IS — specific behavioral traits, not adjectives)
- Worldview (what Leroy believes about chess, competition, learning)

Write to design file: `GAIMER/GameGhost/gaimer_spec_docs/agents/leroy/SOUL.md`

From the builder answers, write the STYLE section (~150 tokens):

Write to design file: `GAIMER/GameGhost/gaimer_spec_docs/agents/leroy/STYLE.md`
Compose into const string in `Agent.cs`

Write to design file: `GAIMER/GameGhost/gaimer_spec_docs/agents/leroy/BEHAVIOR.md`
Compose into const string in `Agent.cs`

Compose into const string in `Agent.cs`

### Task 5: Compose LeroyAntiPatterns const string
From the builder answers + common AI voice failures:

Write to design file: `GAIMER/GameGhost/gaimer_spec_docs/agents/leroy/ANTI-PATTERNS.md`
Compose into const string in `Agent.cs`

### Task 6: Compose BrainPersonalityPrefix for Leroy
Write compact (~200 token) brain-specific personality prefix:
- Identity + analytical style only
- No voice rules, no situations, no anti-patterns

### Task 7: Create calibration examples
Write to design file: `GAIMER/GameGhost/gaimer_spec_docs/agents/leroy/EXAMPLES.md`
- 3-5 good output examples (this is what Leroy sounds like)

### Task 8: Verify token budget
Count tokens for composed personality. Target:

- Flag if any section exceeds budget
