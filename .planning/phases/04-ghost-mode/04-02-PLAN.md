    **MacGhostModeService.cs** (namespace GaimerDesktop.Platforms.MacCatalyst):

    Implements IGhostModeService. Follow the WindowCaptureService pattern for structure.

    Key implementation details:

    - **Constructor:** Call `GhostModeNativeMethods.ghost_panel_create()`. Store result in `_isSupported`. If creation fails, all methods become no-ops (graceful degradation, same pattern as SCK availability check in WindowCaptureService).

    - **GCHandle pinning for callbacks (CRITICAL):** Create two private fields:
      ```
      private GhostModeNativeMethods.GhostModeCallback? _fabTapCallback;
      private GhostModeNativeMethods.GhostModeCallback? _cardDismissCallback;
      private GCHandle _fabTapHandle;
      private GCHandle _cardDismissHandle;
      ```
      In the constructor (after panel creation succeeds), create the managed delegates:
      ```
      _fabTapCallback = () => FabTapped?.Invoke(this, EventArgs.Empty);
      _cardDismissCallback = () => CardDismissed?.Invoke(this, EventArgs.Empty);
      ```
      Pin them with `GCHandle.Alloc(_fabTapCallback)` and `GCHandle.Alloc(_cardDismissCallback)`.
      Register them: `GhostModeNativeMethods.ghost_panel_set_fab_tap_callback(_fabTapCallback)` and same for card dismiss.
      Free handles in Dispose/destructor.

      This prevents the GC from collecting the delegates while native code holds function pointers to them. This is the SAME pattern used in WindowCaptureService for SckCaptureCallback.

    - **EnterGhostModeAsync():**
      1. Set `IsGhostModeActive = true`
      2. Hide the MAUI window: `UIKit.UIApplication.SharedApplication.KeyWindow.Hidden = true;` (wrap in #if MACCATALYST)
      3. Call `GhostModeNativeMethods.ghost_panel_show()`
      4. Return Task.CompletedTask

    - **ExitGhostModeAsync():**
      1. Call `GhostModeNativeMethods.ghost_panel_hide()`
      2. Show the MAUI window: get the UIWindow reference, set `Hidden = false`, call `MakeKeyAndVisible()`
      3. Set `IsGhostModeActive = false`
      4. Return Task.CompletedTask

    - **SetAgentImage(string):** Calls `ghost_panel_set_agent_image(imagePath)`. The C# side should resolve the full filesystem path before calling this (e.g., using NSBundle.MainBundle.PathForResource or the known app bundle path).

    - **SetFabState(bool active, bool connected):** Calls `ghost_panel_set_fab_active(active)` and `ghost_panel_set_fab_connected(connected)`.

    - **ShowCard(FabCardVariant, string?, string?, string?):** Calls `ghost_panel_show_card((int)variant, title, text, imagePath)`.

    - **DismissCard():** Calls `ghost_panel_dismiss_card()`.

    - **Implement IDisposable:** Free GCHandle for both callbacks, call ghost_panel_destroy().
