## Pre-Execution: Download Stockfish for Dev

Before writing integration tests, download the Stockfish binary to the local dev machine:

```bash
# Detect architecture and download
ARCH=$(uname -m)  # arm64 or x86_64
mkdir -p ~/Library/Application\ Support/com.gaimer.witnessdesktop/engines

# Download from official stockfishchess.org (or GitHub releases)
# macOS arm64: stockfish-macos-m1-apple-silicon
# macOS x86_64: stockfish-macos-x86-64-modern
curl -L -o /tmp/stockfish.tar \
  "https://github.com/official-stockfish/Stockfish/releases/download/sf_18/stockfish-macos-m1-apple-silicon.tar"
tar xf /tmp/stockfish.tar -C /tmp/
cp /tmp/stockfish/stockfish-macos-m1-apple-silicon \
  ~/Library/Application\ Support/com.gaimer.witnessdesktop/engines/stockfish
chmod +x ~/Library/Application\ Support/com.gaimer.witnessdesktop/engines/stockfish

# Verify it runs
~/Library/Application\ Support/com.gaimer.witnessdesktop/engines/stockfish <<< "uci" | head -5
```
