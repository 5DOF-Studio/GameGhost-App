```csharp
namespace GaimerDesktop.Services;

public interface IFrameDiffService
{
    /// Computes a 64-bit dHash perceptual hash of the image.
    ulong ComputeHash(byte[] imageData);

    /// Computes hash for a region of interest (ROI) within the image.
    ulong ComputeHash(byte[] imageData, CropRect roi);

    /// Returns the Hamming distance between two hashes (0 = identical).
    int CompareHashes(ulong hash1, ulong hash2);

    /// Returns true if the frame has changed significantly since last check.
    /// Updates internal state with the new hash. Default threshold = 10.
    bool HasChanged(byte[] imageData, int threshold = 10);

    /// Event fired when HasChanged detects a significant change (debounced at 1.5s).
    event EventHandler<FrameChangeEventArgs>? FrameChanged;
}

public record CropRect(int X, int Y, int Width, int Height);

public class FrameChangeEventArgs : EventArgs
{
    public required int HammingDistance { get; init; }
    public required ulong PreviousHash { get; init; }
    public required ulong CurrentHash { get; init; }
    public required DateTime Timestamp { get; init; }
}
```

  <action>
Create FrameDiffService implementing IFrameDiffService. The algorithm is dHash (difference hash) using SkiaSharp which is already a MAUI built-in dependency (no new NuGet needed).

Implementation details:

1. **ComputeHash(byte[] imageData):**
   - Decode with SKBitmap.Decode(imageData). Return 0 on null.
   - Resize to 9x8 grayscale: `bitmap.Resize(new SKImageInfo(9, 8, SKColorType.Gray8), SKFilterQuality.Low)`. Return 0 on null.
   - Get pixel span via `resized.GetPixelSpan()`.
   - Loop: 8 rows x 8 columns. For each pixel at (y*9+x), if pixel > pixel at (y*9+x+1), set bit at position (y*8+x) in a ulong hash.
   - Dispose all SKBitmap with using statements.

2. **ComputeHash(byte[] imageData, CropRect roi):**
   - Decode bitmap, extract subset via `SKRectI(roi.X, roi.Y, roi.X + roi.Width, roi.Y + roi.Height)` and `bitmap.ExtractSubset(cropped, subset)`.
   - Then resize + hash the cropped region identically to the full-frame path.

3. **CompareHashes(ulong hash1, ulong hash2):**
   - Return `BitOperations.PopCount(hash1 ^ hash2)` (requires `using System.Numerics;`).

4. **HasChanged(byte[] imageData, int threshold = 10):**
   - Compute current hash.
   - Calculate Hamming distance from `_lastHash`.
   - If distance >= threshold AND now - _lastChangeTime >= _debounceWindow (1.5 seconds):
     - Update _lastChangeTime, swap _lastHash, fire FrameChanged event, return true.
   - Always update _lastHash with current hash.
   - Return false otherwise.

5. **State fields:**
   - `private ulong _lastHash;`
   - `private DateTime _lastChangeTime = DateTime.MinValue;`
   - `private readonly TimeSpan _debounceWindow = TimeSpan.FromSeconds(1.5);`

Important: Add `using SkiaSharp;` and `using System.Numerics;`. Do NOT add any NuGet packages -- SkiaSharp is already available via MAUI.
